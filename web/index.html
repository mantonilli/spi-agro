¿Cómo funciona?
1
Open-Meteo API entrega precipitación y temperatura diaria desde 1980 para la zona elegida.
2
Se calcula el SPI-30 (objetivo) y SPI-90 (contexto) con ajuste de distribución gamma.
3
El modelo LSTM bidireccional usa 8 variables (incluyendo ONI/ENSO y humedad de suelo) con ventana de 90 días para predecir el SPI-30 a 15 días.
4
Los eventos extremos ponderan 3.25× más que los normales durante el entrenamiento para mejorar la detección de sequías e inundaciones.
